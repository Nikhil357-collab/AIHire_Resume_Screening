# SmartHire AI 🚀

AI-Powered Resume Screening and Candidate Ranking Platform

---

## 📌 Project Overview

SmartHire AI is an Automated Resume Screening Tool built using Python, FastAPI, NLP, and Machine Learning techniques.

The system helps recruiters:
- Upload resumes
- Create job descriptions
- Match resumes with job requirements
- Rank candidates automatically
- Generate ATS scores
- Shortlist top applicants

This project simulates how modern Applicant Tracking Systems (ATS) work in real companies.
